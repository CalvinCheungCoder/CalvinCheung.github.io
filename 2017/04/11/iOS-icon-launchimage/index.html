iOS icon 和 launchimage 尺寸

Posted by Calvin on 2017-04-11

今天对开发的App进行了上架，其中这一版本支持了iPad，所以在打包上传时就需要上传iPad的icon以及launchImage，借此机会把App上架时所需的icon和launchImage的尺寸做一下总结

App Store

1024*1024，仅在iTunes Connect上传一张即可

桌面图标 (app icon)

for iPhone6 Plus(@3x):180 x 180 命名:AppIcon60x60@2x

for iPhone 6/5s/5/4s/4(@2x):120 x 120 命名:AppIcon60x60@3x

for iPad(@1x):76 x 76 命名:AppIcon76x76~ipad

for iPad(@2x):152 x 152 命名:AppIcon76x76@2x~ipad

for iPad Pro:167 x 167 命名:AppIcon83.5x83.5@2x~ipad

系统搜索框图标 (Spotlight search results icon)

for iPhone6 Plus(@3x):120 x 120 命名:AppIcon40x40@3x

for iPhone6/5s/5/4s/4(@2x):80 x 80 命名:AppIcon40x40@2x

for iPad (@1x):40 x 40 命名:AppIcon40x40~ipad

for iPad (@2x):80 x 80 命名:AppIcon40x40@2x~ipad

系统设置图标 (Settings icon)

for iPhone6 Plus(@3x):87 x 87 命名:AppIcon29x29@3x

for iPhone6/5s/5/4s/4(@2x):58 x 58 命名:AppIcon29x29@2x

for iPad (@1x):29 x 29 命名:AppIcon29x29~ipad

for iPad (@2x):58 x 58 命名:AppIcon29x29@2x~ipad

通知栏图标 (Notification)

for iPhone6 Plus(@3x):60 x 60 命名:AppIcon20x20@3x

for iPhone6/5s/5/4s/4(@2x):40 x 40 命名:AppIcon20x20@2x

for iPad (@1x):20 x 20 命名:AppIcon20x20~ipad

for iPad (@2x):40 x 40 命名:AppIcon20x20@2x~ipad

启动图片 (launch image)

普通屏幕（非Retina屏）:320 x 480 命名:LaunchImage.png

for iPhoen5s/5(@2x):640 x 1136 命名:LaunchImage-700-568h@2x

for iPhoen4s/4(@2x):640 x 960 命名:LaunchImage@2x.png

for iPhone 6(@2x):750 x 1334 命名:LaunchImage-800-667h@2x.png

for iPhone 6 Plus(@3x):1242 x 2208 命名:LaunchImage-800-Portrait-736h@3x.png

for iPad（@2x）:1536 x 2048 命名:LaunchImage-700-Portrait@2x~ipad

参考链接:iOS Human Interface Guidelines
